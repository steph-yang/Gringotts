Welcome to Gringotts Digital Currency Exchange Center

Historical Data (Batch Layer)

This section shows historical exchange price of over five thousand digital coins. By typing in amount of holding coins, target coins and holding period, the system will look up in the database and compute a list of exchange price.

Notice:

1. The database only contains historical data from 2020-11-12 to 2020-11-20, because of the large size of files and instability of API. But more data can be scrapped using api.py and real-time data can be found in the second section.

2. Data source and full list of digital currency included can be found here: CoinGecko

Real Time Search (Speed Layer)

This section returns market price of the selected digital coin on the selected day. Data fetched from API will be first processed in the speed layer and then saved into the batch layer database

Notice:

1. The field of coin does not support multiple input. Please type in one coin for each time.

2. Date cannot exceed today.

Client Service

This application is an attempt for a bank-like client service application. Clients can either log in account or register for a new account. When logging into the account, the system will return the client's balance and real-time value of the coins he/she holds.
